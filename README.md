# Typo - A typo helper for python.

Info:
You cannot "pip install typo" you have to put this script in your code and run your code.

> You can also look at example.py to look at  the same example.

## Install:
~~~bash
pip install typo
~~~

## Example:
~~~python
import typo

# All your code is here...
~~~
